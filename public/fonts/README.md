# Font Files

Please upload your custom font files to this directory.

## Required Fonts

### 1. Abordage Regular
Upload one or more of these formats (in order of preference):
- `Abordage-Regular.woff2` (best - recommended)
- `Abordage-Regular.woff`
- `Abordage-Regular.ttf` or `Abordage-Regular.otf`

### 2. Meridien LT Std Roman
Upload one or more of these formats (in order of preference):
- `MeridienLTStd-Roman.woff2` (best - recommended)
- `MeridienLTStd-Roman.woff`
- `MeridienLTStd-Roman.ttf` or `MeridienLTStd-Roman.otf`

## How to Upload

1. In VS Code, navigate to this folder: `public/fonts/`
2. Drag and drop your font files into this directory
3. Make sure the file names match the names above (or let me know if they're different)

## Converting Fonts (if needed)

If you only have `.ttf` or `.otf` files, I recommend converting to `.woff2` for better performance:
- Use: https://cloudconvert.com/ttf-to-woff2
- Or: https://www.fontsquirrel.com/tools/webfont-generator

## Fallback Fonts

Don't worry if you don't have the fonts immediately - the website will use fallback fonts (Georgia for Abordage, Times New Roman for Meridien) until the custom fonts are uploaded.
